## Main Idea

2026年4月 GitHub 上5个自我进化记忆 Agent 项目暴火，形成从自我记忆进化到多 Agent 辩论、自动科研、生产部署的完整链路。核心趋势：AI 不再是工具，而是能自己长大、自己吵架、自己干活的"创业+科研全栈合伙人"。

## Key Details

**5个爆款项目：**

1. **NousResearch/hermes-agent** (24h +8800★)
   - 自我进化记忆 Agent
   - 动态 patch 机制让 AI 伴随用户成长，越用越懂用户偏好
   - 解决传统 Agent"一次忘光光"的痛点

2. **TauricResearch/TradingAgents** (24h +3900★)
   - 多 Agent 模拟完整交易公司
   - 研究员挖数据、风控喊停、决策者互相撕逼后再下单
   - 一干掉单 Agent 拍脑袋决策的血亏风险

3. **SakanaAI/AI-Scientist-v2** (24h +2000★)
   - Agentic tree search 驱动的全自动科研 Agent
   - 从提假设、跑实验到写论文一条龙
   - 一个人顶10人实验室

4. **microsoft/agent-framework** (24h +1800★)
   - 微软官方 Agent 编排框架
   - 支持 Python + .NET
   - 一键搭建、生产部署多 Agent 工作流

5. **block/goose** (24h +1500★)
   - 开源可扩展 AI Agent
   - 超越代码补全，能自主安装、执行、编辑、测试任意 LLM 任务
